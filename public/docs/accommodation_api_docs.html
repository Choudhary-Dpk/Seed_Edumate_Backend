GET /accom/properties

Get property listings with optional filters

Query Parameters:

Parameter Type Required Description
city string Optional Filter by city name (e.g., "London")
provider string Optional Filter by provider name (e.g., "iQ")
universityId string Optional Filter by university ID (24 chars)

  -H "X-API-Key: your-api-key"

üìù Lead Management APIs

POST /accom/leads

Create a new accommodation lead

Request Body Fields:

Field Type Required Constraints
firstName string Required Max 25 characters
lastName string Required Max 25 characters
email string Required Valid email, max 52 chars
contactNo string Required Format: +91-9999999999
dateOfBirth string Required Format: YYYY-MM-DD
budget string Required Format: CURRENCY-AMOUNT
gender string Required Max 6 characters
nationality string Required -
universityId string Required 24 characters
message string Optional Max 255 characters 
curl -X POST https://api.edumateglobal.com//accom/leads \
  -H "X-API-Key: your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "firstName": "John",
    "lastName": "Doe",
    "email": "john.doe@example.com",
    "contactNo": "+91-9999999999",
    "dateOfBirth": "2000-01-01",
    "budget": "INR-10000",
    "gender": "male",
    "nationality": "Indian",
    "universityId": "5c132a7d3ef96e7a40d6a7a7",
    "message": "Looking for accommodation"
  }'
